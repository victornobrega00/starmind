⭐ STARMIND — Plataforma de Streaming Educativo e Gamificação
O StarMind é uma plataforma inovadora de streaming educativo voltada para o público infantil. O sistema une aprendizado e entretenimento através de uma interface totalmente gamificada, oferecendo um ambiente seguro, controlado e em estrita conformidade com as diretrizes de proteção à criança no ambiente digital.

🚀 Funcionalidades Principais
🛡️ Central do Responsável (Controle Parental)
Painel de Controle: Um ambiente exclusivo para os pais gerenciarem a experiência dos filhos, permitindo a adição e exclusão de conteúdos didáticos.

Controle de Tempo de Tela: Slider dinâmico e intuitivo que permite definir um limite diário de uso (de 15 minutos a 4 horas).

Segurança por PIN: O acesso à área dos pais é protegido por um PIN numérico de 4 dígitos para evitar alterações indesejadas pelas crianças.

⚖️ Conformidade com o ECA Digital (Lei Felca)
O projeto foi estruturado seguindo o modelo de responsabilidade compartilhada proposto pela Lei N.º 15.211/2025.

Anúncios Comerciais Zero: Garante uma experiência limpa e focada no aprendizado.

Navegação Blindada: Bloqueio automático contra hiperlinks externos inseguros.

🎮 Interface Infantil e Aprendizado Gamificado
Filtro por Disciplinas: Navegação fluida por matérias através de um sistema de cards temáticos (Matemática, Ciências, Português, Inglês, Arte e Música).

Sistema de Recompensas (Stars): Cada vídeo assistido concede uma pontuação em estrelas (ex: ⭐ +50), incentivando o engajamento do aluno.

Status de Progresso: Identificação visual imediata de aulas já assistidas através de badges dinâmicos (✔ Assistido).

Identidade Visual por Canal: Tags customizadas que indicam visualmente a origem de cada conteúdo (YouTube Kids, Netflix, etc.) com posicionamento fixo inteligente e design moderno.

🛠️ Tecnologias Utilizadas
O ecossistema do StarMind foi construído utilizando tecnologias modernas e focadas em desempenho e organização arquitetural:

Frontend
HTML5: Estrutura semântica organizada para acessibilidade e clareza de componentes.

CSS3 Avançado:

Uso de Flexbox e CSS Grid para layouts fluidos, responsivos e centralização perfeita de telas.

Design baseado em componentes modernos (Pill design, sombras suaves e efeitos de Glassmorphism com backdrop-filter).

Tipografia integrada com a fonte Nunito via Google Fonts para leitura infantil facilitada.

JavaScript (ES6): Manipulação dinâmica do DOM, renderização de listas com fallbacks de segurança anti-undefined, tratamento de strings contra falhas de injeção e controle de fluxo por estados (AppState).

Backend 
Java & Spring Boot: Arquitetura robusta para o gerenciamento de regras de negócio, persistência de dados e segurança da API.
